## 2.17.0

### Android

* Gamebase Android SDK 2.17.0 에서 Gamebase.ImageNotice.showImageNotices API를 호출하면 크래시가 발생합니다.
    * 2.17.0 의 크래시 및 OS 5.0~6.0 에서 커스텀 스킴 이벤트가 동작하지 않는 이슈가 수정된 Gamebase Android SDK 2.17.4 를 사용하시기 바랍니다.
