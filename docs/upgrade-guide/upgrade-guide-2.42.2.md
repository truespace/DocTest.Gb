## 2.42.2

### Unity

* Gamebase Setting Tool (v2.5.0)에 ONE Store v19 결제 어댑터가 추가되었습니다.
    * **SettingTool > Android** 설정에서 ONE Store v19 어댑터 활성화 시 iap_sdk-v19.xx.xx.aar 다운로드 페이지로 연결되며, 해당 파일을 **Assets > Plugins > Android** 폴더에 복사해야 합니다.
