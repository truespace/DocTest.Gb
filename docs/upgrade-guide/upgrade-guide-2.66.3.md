## 2.66.3

### Unity

#### Changed Minimum Support Version

* 최소 지원 Unity 버전이 2018.4.0에서 2020.3.0으로 변경되었습니다.
* 하위 버전의 Unity 지원이 필요하다면 [고객 센터](https://toast.com/support/inquiry)로 문의해 주시기 바랍니다.
