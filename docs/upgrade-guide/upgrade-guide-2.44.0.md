## 2.44.0

### Android

* Gamebase Android SDK 2.44.0에서 registerPush를 호출하면 Android 6.0(M, API Level 23) 단말기에서 크래시가 발생합니다.
    * 문제가 해결된 Gamebase Android SDK 2.44.1을 사용하세요.
