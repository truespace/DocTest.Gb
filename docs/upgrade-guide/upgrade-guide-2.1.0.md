## 2.1.0

### Common

#### Removed APIs

* 사용되지 않는 TransferKey 기능이 제거되었습니다.
	* Guest 계정 이전 기능이 필요하다면 Gamebase SDK 2.2.1부터 추가된 [\[Game > Gamebase > Unity SDK 사용 가이드 > 인증 > TransferAccount\]](../unity-authentication.md#transferaccount) 기능을 이용하세요.
