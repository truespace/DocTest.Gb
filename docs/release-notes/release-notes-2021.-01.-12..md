### 2021. 01. 12.

```
Gamebase의 XCode 최소 지원 버전이 10에서 11로 변경되었습니다. 
```

#### 기능 추가
* [Console] 푸시 신규 메뉴 추가
	* 통계: 푸시 발송, 수신, 토큰 등록 등의 푸시 통계를 확인
	* 이벤트 키: 푸시 발송 통계에 사용하는 이벤트 키를 관리
	* 인증서: 푸시 발송에 사용하는 인증서를 관리
	* 설정: 푸시 관련된 설정값을 관리
