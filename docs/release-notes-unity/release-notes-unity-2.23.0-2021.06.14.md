### 2.23.0 (2021.06.14)

* 외부 SDK 업데이트: TOAST Unity SDK(0.22.1)
* Unity 2020.2 이후 버전에서 발생하는 Warning 제거
* Standalone과 Unity Editor에서 초기화 속도 개선

#### 버그 수정
* 약관 동의를 했음에도 ShowTermsView API 호출하면 PushConfiguration 결과가 null이 아닌 문제 수정
