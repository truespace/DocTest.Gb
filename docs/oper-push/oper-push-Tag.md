### Tag detail

등록된 태그의 관리 및 해당 태그에 등록된 유저의 목록을 관리할 수 있습니다.
![push_18](./image/push_18_kr_240103.png)
<!-- LLM_Image_DESC_20260406
    유형: Screenshot
    내용: Gamebase Push - 태그 상세 정보 화면
    구성: '태그 상세 정보' 제목 아래에 삭제/수정 버튼이 있음. ID, 이름(tag_test1), 등록일, 수정일이 표시됨. 하단에 '유저 ID 관리' 섹션이 있으며, 삭제/등록 버튼과 No, 유저 ID, 등록일 컬럼으로 구성된 유저 목록 테이블이 배치됨. 이전/다음 페이지 버튼이 있음
    Keyword: 태그 상세, 유저 ID 관리, 등록, 삭제, 유저 목록
-->

상단의 **삭제**, **수정** 버튼을 클릭해 태그 정보를 수정하거나 삭제할 수 있으며 하단의 유저 ID 관리 기능을 사용해 태그에 유저를 등록하거나 삭제할 수 있습니다.

#### 유저 등록

##### 단건 등록
![push_19](./image/push_19_kr_240103.png)

##### 파일 등록

![push_20](./image/push_20_kr_240103.png)
<!-- LLM_Image_DESC_20260406
    유형: UI
    내용: Gamebase Push - 태그 유저 ID 등록 팝업 (파일 업로드)
    구성: '태그 유저 ID 등록' 제목 아래에 입력 방법(파일 선택), 파일 선택/템플릿 예시 파일 링크가 있음. 드래그앤드롭 파일 업로드 영역과 파일명/파일 크기 표시가 있음. 최대 1,000명까지 등록 가능하다는 안내와 하단에 닫기/등록 버튼이 배치됨
    Keyword: 유저 ID 등록, 파일 업로드, 태그, 대량 등록
-->

**등록** 버튼을 클릭하면 위와 같이 등록 팝업이 나타나며 직접 ID를 입력하거나 파일을 등록해 입력할 수 있습니다.

**파일 등록**으로는 한 번에 최대 1,000명까지 등록할 수 있습니다.

#### 유저 삭제

![push_21](./image/push_21_kr_240103.png)
<!-- LLM_Image_DESC_20260406
    유형: Screenshot
    내용: Gamebase Push - 유저 ID 관리 화면 (삭제)
    구성: '유저 ID 관리' 제목 아래에 삭제/등록 버튼이 있음. 왼쪽에 체크박스가 있는 유저 목록 테이블(No, 유저 ID, 등록일)이 배치됨. 체크박스를 선택하여 유저를 삭제할 수 있음. 이전/다음 페이지 버튼이 있음
    Keyword: 유저 ID 관리, 삭제, 체크박스, 유저 목록
-->

태그에 등록된 유저를 삭제하려면 유저 목록에서 왼쪽의 체크 박스를 선택한 후 **삭제** 버튼을 클릭합니다.
