## Open External Browser

다음 API를 통하여 외부 브라우져를 열 수 있습니다. 파라미터로 전송되는 URL은 유효한 값이어야 합니다.

**API**

```java
+ (void)Gamebase.WebView.openWebBrowser(Activity activity, String urlString);
```
