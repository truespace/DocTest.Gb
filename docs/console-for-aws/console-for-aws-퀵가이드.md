## Game > Gamebase > Console for AWS

AWS marketplace를 통해 회원가입을 한 사용자가 로그인하는 콘솔의 기본적인 설정과 사용 방법을 안내합니다.

Gamebase for AWS 의 콘솔은 아래와 같은 기능을 제공합니다.
* 프로젝트, 서비스 관리
* 서비스를 이용하는 회원 관리
