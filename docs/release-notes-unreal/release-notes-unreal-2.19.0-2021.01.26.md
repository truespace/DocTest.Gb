* SDK 배포: 2.16.0 ~ 2.19.0 누적된 내역 반영

    * 고객 센터 기능 추가    
    * 인증 추가: Hangame, Weibo
    * Galaxy 스토어 추가
    * 결제 아이템 정보에 지역화된 상품 정보 추가: localizedTitle, localizedDescription
    * Android 설정 툴 제공
    * Unreal 4.26 지원
