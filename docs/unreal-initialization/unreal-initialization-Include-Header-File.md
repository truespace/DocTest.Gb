## Game > Gamebase > Unreal SDK 사용 가이드 > 초기화

Gamebase Unreal SDK를 사용하려면 먼저 초기화를 진행해야 합니다. 또한 앱 ID, 앱 버전 정보가 NHN Cloud Console에 반드시 등록돼 있어야 합니다.


### Include Header File

Gamebase API를 사용하기 위해서는 다음의 헤더 파일을 가져옵니다.

```cpp
#include "GamebaseSubsystem.h"
```
