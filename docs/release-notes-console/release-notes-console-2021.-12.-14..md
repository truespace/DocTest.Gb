### 2021. 12. 14.

#### 기능 개선/변경
* 앱 > 앱 > 고객센터
	* 웹 브라우저에서 고객센터에 접속하여 바로 1:1 문의할 수 있는 기능 추가
* 운영 > 점검
	* 점검 페이지에서 점검 시간 표시 여부를 선택하는 기능 추가
* 고객센터 > 템플릿
	* 기존에 등록된 템플릿 중 고객센터에서 더 이상 지원하지 않는 언어로 작성된 템플릿 미노출 및 삭제

#### 버그 수정
* 앱 > 앱 > 고객센터
	* 지원 언어에 중국어를 설정하였어도 고객센터에서 중국어가 표시되지 않는 오류 수정
